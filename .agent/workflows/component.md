---
description: Create a generic React component
---
1. **Goal**: Create a reusable UI component.
2. **Prerequisites**:
   - Component Name.
   - Functionality (what does it do?).
   - Props interface.
3. **Standards**:
   - Use `vercel-react-best-practices`.
   - Use `web-design-guidelines` for UI/UX.
4. **Implementation Steps**:
   - Create file in `frontend/src/components/ui/` or `frontend/src/components/`.
   - Use Functional Components with TypeScript (`React.FC` or direct typing).
   - **Styles**: Use Tailwind CSS. ensuring responsiveness and dark mode support if applicable.
   - **Accessibility**: Ensure ARIA attributes are used where necessary (e.g., for custom controls).
