1. **Goal**: Encapsulate logic in a reusable React Hook.
2. **Prerequisites**:
   - Hook Name (must start with `use`).
   - Logic description.
3. **Standards**:
   - Use `vercel-react-best-practices` (memoization, dependency arrays).
4. **Implementation Steps**:
   - Create file in `frontend/src/hooks/`.
   - Ensure the hook handles cleanup (e.g., `useEffect` return).
   - Return a typed object/tuple.
   - Write a simple usage example in the comments.
