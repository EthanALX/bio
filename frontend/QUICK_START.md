# 🚀 快速启动指南

## 立即查看效果

### 1. 启动开发服务器
```bash
cd frontend
pnpm run dev
```

### 2. 打开浏览器
- **主页**: http://localhost:3000
- **3D 地球演示**: http://localhost:3000/globe

## 你会看到什么

### 主页 (/)
1. **动态粒子背景** - 移动鼠标,粒子会跟随你的光标
2. **增强版统计卡片** - 左侧边栏,悬停查看发光效果
3. **增强版日历** - 点击 "Days" 卡片切换到日历视图
   - 悬停日期格子查看详细信息
   - 观察连击连接线和最佳连击标记

### 3D 地球页 (/globe)
1. **旋转地球** - 自动旋转展示
2. **拖拽交互** - 鼠标拖拽旋转地球
3. **滚轮缩放** - 放大/缩小查看细节
4. **路线可视化** - 3D 路线曲线(需要有坐标数据)

## 主要交互

### 统计卡片
- **悬停**: 卡片向左移动,边框发光
- **点击**: 波纹扩散,切换视图
- **激活**: 持续发光 + 背景粒子

### 日历热力图
- **悬停格子**: 显示详细 tooltip
- **颜色强度**: 5 级,表示跑步距离
- **连击线**: 连接连续跑步的天数
- **最佳连击**: 金色高亮 + 火焰图标

### 3D 地球
- **拖拽**: 旋转地球
- **滚轮**: 缩放视角
- **自动旋转**: 缓慢自转展示

## 可选配置

### Mapbox 地图 (可选)
如果你想使用增强版地图功能:

1. 注册 Mapbox 账号: https://www.mapbox.com/
2. 获取 Access Token
3. 创建 `.env.local` 文件:
```env
NEXT_PUBLIC_MAPBOX_TOKEN=your_token_here
```
4. 重启开发服务器

## 性能说明

### 自动性能检测
系统会自动检测你的设备性能并调整效果:

- **高端设备**: 5000 粒子 + 全部特效
- **中端设备**: 2000 粒子 + 部分特效
- **低端设备**: 500 粒子或纯 CSS

### 推荐浏览器
- Chrome 90+
- Edge 90+
- Safari 14+
- Firefox 88+

## 常见问题

### Q: 粒子效果卡顿?
A: 系统会自动降级,如果仍然卡顿,可以在 `BackgroundOrchestrator.tsx` 中手动减少粒子数量。

### Q: 3D 地球显示空白?
A: 确保浏览器支持 WebGL。打开控制台检查是否有错误信息。

### Q: 地图不显示?
A: 需要配置 Mapbox Token。如果不需要地图功能,可以忽略。

### Q: 构建失败?
A: 运行 `pnpm install` 确保所有依赖已安装。
